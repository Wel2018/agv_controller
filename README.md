# AGV 遥操作调试程序

底盘遥操作控制，主要适配了云迹 Water 底盘。

> [!note]
> api 文档和使用手册、底盘模型文件见 `docs` 文件夹。

## 特性

- 支持键盘按键遥控
- 支持定点导航
- 支持快速软急停或取消动作
